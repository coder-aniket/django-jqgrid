Quick Start Guide

1. Installation
pip install django-jqgrid
2. Add to INSTALLED_APPS
INSTALLED_APPS = [
    # ... your other apps
    'rest_framework',
    'django_jqgrid',
]
3. Basic Usage
{% templatetag openvariable %} load jqgrid_tags {% templatetag closevariable %}

{% templatetag openvariable %} jqgrid "myGrid" api_url="/api/data/" {% templatetag closevariable %}
4. Configure ViewSet
class MyModelViewSet(viewsets.ModelViewSet):
    queryset = MyModel.objects.all()
    serializer_class = MyModelSerializer
